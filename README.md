# ArtGateOne-DMX
This is simple artnet node.


Required:
Arduino UNO

Ethernet Shield with W5100/W5500

OLED display 128x32

module with max485

---------------------------------------

U can use web browser to config settings

U can record static scene.

---------------------------------------

WIRING

OLED DISPLAY

VCC --> 5v or 3.3v

GND --> GND

SCL --> SCL

SDA --> SDA

----------
MAX485 module

RO --> not connnected

RE + DE + VCC --> 5v

ro --> TX (D1 PIN)

GND --> GND

A --> DMX OUT

B --> DMX OUT
